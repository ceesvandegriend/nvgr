=======================================
nvgr / navigare - Nautical Calculations
=======================================

``nvgr`` is a Python module for nautical calculations.

It uses:
   - latitude in degrees North or South

   - longitude in degrees East or West

   - course in degrees, North is 000° or 360°, East is 090°, South is 180° and West is 270°

   - distance in nautical miles, 1/60 of a degree.

.. warning::
   The code and documentation is work in progress.
